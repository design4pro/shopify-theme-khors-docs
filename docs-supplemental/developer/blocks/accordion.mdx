### \_accordion-item Block

`blocks/_accordion-item.liquid`

This is a private block (prefixed with `_`), meaning it cannot be added independently by merchants. It is only available as a child of the accordion block.

#### LiquidDoc

```liquid
{% doc %}
  Private accordion item block, used as a nested child of the accordion block.
  Renders a details/summary element with heading, content, and toggle icon.
  Inherits icon style, heading size, and content size from the parent accordion via CSS.

  @example
  Used automatically via {% content_for 'blocks' %} inside accordion.liquid
{% enddoc %}
```

## CSS Classes

- `.accordion` - Main container
- `.accordion--chevron` - Chevron icon style modifier
- `.accordion--plus-minus` - Plus/minus icon style modifier
- `.accordion--bordered` - Bordered variant with top/bottom borders
- `.accordion__item` - Item container (`details` element)
- `.accordion__header` - Summary/trigger element
- `.accordion__title` - Heading text
- `.accordion__icon` - Icon container with rotation transition
- `.accordion__icon-chevron` - Chevron icon variant
- `.accordion__icon-plus` - Plus icon variant
- `.accordion__icon-minus` - Minus icon variant
- `.accordion__content-wrapper` - Animated content wrapper (CSS grid trick for smooth expand/collapse)
- `.accordion__content` - Content area
- `.accordion__empty` - Empty state message

## JavaScript

The `AccordionSingleOpen` class handles single-open mode. When enabled via the `data-accordion-single-open` attribute, opening one accordion item automatically closes all others.

```javascript
class AccordionSingleOpen {
  constructor(container) {
    this.container = container;
    this.details = container.querySelectorAll('details');

    for (const detail of this.details) {
      detail.addEventListener('toggle', (event) => {
        if (event.target.open) {
          this.#closeOthers(event.target);
        }
      });
    }
  }

  #closeOthers(openDetail) {
    for (const detail of this.details) {
      if (detail !== openDetail && detail.open) {
        detail.open = false;
      }
    }
  }
}
```
