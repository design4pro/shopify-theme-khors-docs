## Location

`blocks/heading.liquid`

## LiquidDoc

```liquid
{% doc %}
  Heading block for Shopify Theme Editor.
  Provides semantic HTML headings (h1-h6 + display variant) with flexible styling options.

  Features:
  - 7 heading levels (h1-h6 + display) mapped to typography scale
  - Optional subheading text
  - Text alignment (left, center, right)
  - Visual decorations (underline, accent border)
  - Color scheme override
  - Configurable spacing (margin top/bottom)

  @example Default usage in Theme Editor
    Add this block to any section that accepts @theme blocks
{% enddoc %}
```

- `.heading--accent` - Accent border decoration modifier
