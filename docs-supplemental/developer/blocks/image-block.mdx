## Location

## LiquidDoc

```liquid
{% doc %}
  Image block for Shopify Theme Editor.
  Renders a responsive image with configurable aspect ratio, lazy loading,
  overlay, caption, and optional link wrapper.

  @example Default usage in Theme Editor
    Add this block to any section that accepts @theme blocks
{% enddoc %}
```

## CSS Classes

- `.image-block` - Main container (figure element)
- `.image-block__media` - Media wrapper
- `.image-block__img` - Image element
- `.image-block__img--cover` - Cover object fit modifier
- `.image-block__img--contain` - Contain object fit modifier
- `.image-block--16-9` - 16:9 aspect ratio modifier
- `.image-block--4-3` - 4:3 aspect ratio modifier
- `.image-block--3-2` - 3:2 aspect ratio modifier
- `.image-block--1-1` - 1:1 aspect ratio modifier
- `.image-block--2-3` - 2:3 aspect ratio modifier
- `.image-block--3-4` - 3:4 aspect ratio modifier
- `.image-block--9-16` - 9:16 aspect ratio modifier
- `.image-block__overlay` - Color overlay
- `.image-block__link` - Optional link wrapper
- `.image-block__caption` - Caption element
- `.image-block__caption--left` - Left caption alignment
- `.image-block__caption--center` - Center caption alignment
- `.image-block__caption--right` - Right caption alignment
- `.image-block--placeholder` - Placeholder state
- `.image-block__placeholder-svg` - Placeholder SVG
