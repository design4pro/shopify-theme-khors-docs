## LiquidDoc

```liquid
{% doc %}
  Icon block wrapper for Shopify Theme Editor.
  Delegates rendering to the icon snippet while providing merchant-friendly UI.

  @example Default usage in Theme Editor
    Add this block to any section that accepts @theme blocks
{% enddoc %}
```

## CSS Classes

- `.icon-link` - Link wrapper when URL is provided
- `.icon-link:hover` - Reduces opacity on hover for visual feedback
- `.icon-link:focus-visible` - Accent outline with offset for keyboard navigation
