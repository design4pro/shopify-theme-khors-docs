## Overview

The cart section renders the `/cart` page with an overview of items in the customer's cart. It provides a basic cart form with product images, titles, quantity inputs, and a checkout button.

This is a minimal implementation using a `<table>` layout with the Shopify `cart` object.

## Schema

```json
{
  "name": "t:general.cart",
  "settings": []
}
```

<Note>
  The cart section currently has no configurable settings. It renders a standard cart table with image, title, quantity, and checkout controls.
</Note>

## Template Usage

Used in `templates/cart.json`:

```json
{
  "sections": {
    "main": {
      "type": "cart"
    }
  }
}
```

## Rendered Structure

```html
<h1>Your cart</h1>
<form action="/cart" method="post">
  <table>
    <tr>
      <td><!-- product image via image snippet --></td>
      <td>
        <p>Product Title</p>
        <a href="/cart/change?...">Remove</a>
      </td>
      <td>
        <input type="text" name="updates[]" value="1">
        <input type="submit" value="Update">
      </td>
    </tr>
  </table>
  <input type="submit" name="checkout" value="Check out">
</form>
```
