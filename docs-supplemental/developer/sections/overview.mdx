## What are Sections?

Sections are full-width, customizable modules that make up your theme pages. Each section has a `{% schema %}` block defining its settings.

## Section Structure

```liquid
{% comment %} Section HTML/Liquid {% endcomment %}
<section class="section-name">
  {{ section.settings.heading }}
</section>

{% schema %}
{
  "name": "t:sections.section_name.name",
  "settings": [
    {
      "type": "text",
      "id": "heading",
      "label": "t:labels.heading"
    }
  ],
  "blocks": [{ "type": "@theme" }],
  "presets": [
    {
      "name": "t:sections.section_name.presets.default"
    }
  ]
}
{% endschema %}
```
