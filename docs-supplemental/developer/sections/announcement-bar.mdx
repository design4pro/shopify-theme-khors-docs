## Architecture

A simple, self-contained section with inline JavaScript. Part of `sections/header-group.json` and renders above the header.

- Rich text content with optional link
- Dismissible via localStorage persistence
- Color scheme support
- No external dependencies

## Schema Settings

| Setting | Type | Default | Description |
|---------|------|---------|-------------|
| `text` | `richtext` | Welcome message | Announcement text content |

## JavaScript

The `AnnouncementBar` class (inline via `{% javascript %}`) handles dismissal:

```js
// Storage key format
`khors:announcement-dismissed:${sectionId}`
```

- Checks `localStorage` on init — hides if previously dismissed
- On dismiss click: sets `hidden` attribute and stores to `localStorage`
- Gracefully handles private browsing (session-only dismiss on `catch`)
- Uses ES6 private methods (`#isDismissed`, `#dismiss`)

## Accessibility

- `role="region"` with `aria-label` on root
- Dismiss button has `aria-label` translation key
- `focus-visible` outline on dismiss button
- `prefers-reduced-motion` disables transition on dismiss button
