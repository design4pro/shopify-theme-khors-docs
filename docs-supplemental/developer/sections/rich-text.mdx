## Location

`sections/rich-text.liquid`

## Overview

The rich text section provides a flexible canvas for formatted content including headings, paragraphs, lists, links, bold/italic text, and more. It supports three width options and customizable text alignment for versatile page layout creation.

## Architecture

This is a single-field section that renders Shopify's richtext editor output with pre-styled typography hierarchy and spacing.

| Feature | Description |
|---------|-------------|
| Content Width | Narrow (800px), Medium (1000px), or Full (page-width) |
| Text Alignment | Left, Center, or Right |
| Typography | Styled h2-h6, paragraphs, lists, links |
| Color Schemes | Light, Dark, or Accent |

## Typography Hierarchy

All heading and text elements inside `.rich-text-section__content` are automatically styled:

| Element | Mobile Size | Desktop Size | Weight | Notes |
|---------|-------------|--------------|--------|-------|
| `h2` | 2xl | 3xl | Semibold | Primary heading |
| `h3` | xl | 2xl | Semibold | Subheading |
| `h4` | lg | xl | Semibold | Tertiary heading |
| `h5` | base | lg | Semibold | Minor heading |
| `h6` | sm | base | Semibold | Uppercase, letterspaced |
| `p` | base | lg | Normal | Body text |
| `a` | base | lg | Normal | Accent color, underline on hover |
| `ul/ol` | base | lg | Normal | Disc/decimal markers |

Spacing follows a consistent vertical rhythm with CSS custom properties.

## Supported Rich Text Features

The Shopify richtext editor supports:

- **Headings**: H2, H3, H4, H5, H6
- **Text formatting**: Bold, italic, underline
- **Links**: Internal and external
- **Lists**: Ordered (numbered) and unordered (bulleted)
- **Alignment**: Left, center, right (controlled by section setting, not editor)

<Note>
H1 is intentionally excluded to preserve semantic page hierarchy. Use H2 as the top-level heading within rich text sections.
</Note>

## Width Options

| Width | Max Width | Use Case |
|-------|-----------|----------|
| Narrow | 800px | Long-form reading (blog-style content) |
| Medium | 1000px | Balanced content (about pages, policies) |
| Full | Page width (1280px) | Full-bleed content or multi-column layouts |

## Empty State

If no content is provided, displays placeholder text: "Add content using the richtext editor."

## Accessibility

- Section includes `role="region"` with translated `aria-label`
- Typography hierarchy uses semantic HTML (h2-h6, p, ul, ol)
- Links have sufficient color contrast and visible focus states
- Line height set to `--line-height-relaxed` (1.7) for readability
- `prefers-reduced-motion` respected for link transitions
