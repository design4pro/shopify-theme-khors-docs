## Architecture

The footer uses a two-zone layout:

- **Top zone** — Grid of content columns (menu/text blocks) plus optional newsletter
- **Bottom bar** — Social links, copyright, and payment icons

| Component | File | Role |
|-----------|------|------|
| [Newsletter](/developer/snippets/footer-newsletter) | `footer-newsletter.liquid` | Email signup with honeypot |
| [Social Links](/developer/snippets/footer-social) | `footer-social.liquid` | Platform icons from global settings |
| [Payment Icons](/developer/snippets/footer-payment) | `footer-payment.liquid` | Enabled payment type SVGs |

Included in `sections/footer-group.json` as the sole section.

## Blocks

### Menu Block (`menu`)

| Setting | Type | Description |
|---------|------|-------------|
| `heading` | `text` | Column heading |
| `menu` | `link_list` | Navigation menu to display |

### Text Block (`text`)

| Setting | Type | Description |
|---------|------|-------------|
| `heading` | `text` | Column heading |
| `content` | `richtext` | Rich text content |

## Schema Settings

| Setting | Type | Default | Description |
|---------|------|---------|-------------|
| `show_newsletter` | `checkbox` | `true` | Show newsletter signup |
| `newsletter_heading` | `text` | `"Stay in touch"` | Newsletter section heading |
| `show_social` | `checkbox` | `true` | Show social media icons |
| `show_payment_icons` | `checkbox` | `true` | Show payment method badges |
| `show_copyright` | `checkbox` | `true` | Show copyright line |
| `show_border` | `checkbox` | `true` | Top border line |
| `color_scheme` | `select` | `light` | `light`, `dark`, `accent` |
| `padding_top` | `range` | `40` | Top padding 0–80px, step 4 |
| `padding_bottom` | `range` | `40` | Bottom padding 0–80px, step 4 |

## Accessibility

- Menu links use `role="list"` for screen reader announcement
- Links have `focus-visible` outlines
- Copyright contains shop name as a link to root
- `prefers-reduced-motion` disables transitions
- Mobile: columns stack to single column, bottom bar centers
