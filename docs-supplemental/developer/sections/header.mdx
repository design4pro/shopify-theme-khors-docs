## Architecture

The header section orchestrates seven sub-components via `{% render %}` plus a shared JavaScript module:

| Component | File | Role |
|-----------|------|------|
| [Logo](/developer/snippets/header-logo) | `header-logo.liquid` | Responsive logo with retina support |
| [Navigation](/developer/snippets/header-nav) | `header-nav.liquid` | 3-level desktop menubar + mobile accordion |
| [Mega Menu](/developer/snippets/header-mega-menu) | `header-mega-menu.liquid` | Full-width dropdown with featured image |
| [Dropdown](/developer/snippets/header-nav-dropdown) | `header-nav-dropdown.liquid` | Recursive L2/L3 narrow dropdown |
| [Drawer](/developer/snippets/header-drawer) | `header-drawer.liquid` | Mobile slide-in dialog |
| [Search](/developer/snippets/header-search) | `header-search.liquid` | Predictive search overlay |
| [Cart Icon](/developer/snippets/header-cart-icon) | `header-cart-icon.liquid` | Dynamic badge with live count |
| Sticky + Cart JS | `assets/header.js` | StickyHeader + CartCount classes |

The section is included in `sections/header-group.json` alongside the [Announcement Bar](/developer/sections/announcement-bar).

## Layout Variants

Three layout modes controlled by `section.settings.layout`:

| Value | Description |
|-------|-------------|
| `logo-left-nav-center` | Logo left, nav centered, icons right (default) |
| `logo-center-nav-below` | Logo centered on full row, nav below, icons absolute-right |
| `logo-left-nav-left` | Logo and nav left-aligned, icons right |

Each layout variant applies CSS via `.header--{layout}` modifier classes on the root `<header>` element.

## JavaScript

The `assets/header.js` file (≤2KB gzipped) provides two classes:

### StickyHeader

Uses `IntersectionObserver` on a sentinel element to toggle `.header--stuck` class and apply `position: fixed` without scroll event listeners. Opt-in via `data-sticky="true"` attribute.

### CartCount

Listens for `shopify:section:load` and cart form submissions, fetches `/cart.json`, and dispatches `khors:cart:updated` custom event with `{ count, cart }` detail.

Snippet-specific JS (drawer, navigation, mega menu, search, cart icon, announcement bar) is embedded inline via `{% javascript %}` for Shopify's lazy-loading optimization.

## Accessibility

- Mobile hamburger button uses `aria-label` with translation key
- Desktop navigation follows ARIA menubar pattern
- Noscript fallback renders flat link list for no-JS
- All icon buttons have 44×44px minimum touch target
- `focus-visible` outlines on all interactive elements
- `prefers-reduced-motion` disables transitions
