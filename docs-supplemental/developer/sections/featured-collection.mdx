## Overview

The featured collection section showcases products from a chosen Shopify collection in a responsive grid layout. It supports an optional banner header with overlay text, configurable product display settings, and a "View All" button linking to the full collection page.

## Architecture

This section renders products using the `product-card` snippet and provides two distinct visual areas:

| Component | Description |
|-----------|-------------|
| Banner (optional) | Collection featured image with optional overlay and title |
| Product Grid | Responsive grid of product cards with customizable columns |
| View All Button | Optional link to full collection page |

The section integrates with Shopify's collection object to access products, title, description, and featured image.

## Image Loading Strategy

The section implements progressive image loading for optimal performance:

- Banner image: `loading="eager"` with `fetchpriority="high"` (above the fold)
- First 4 product images: `loading="eager"` (likely visible on initial load)
- Remaining product images: `loading="lazy"` (deferred loading)

Banner uses responsive `srcset` at widths: 600w, 1200w, 1800w, 2400w.

## Responsive Behavior

Grid columns adapt based on viewport:

| Viewport | Columns | Gap |
|----------|---------|-----|
| Mobile (<48em) | `columns_mobile` (1–2) | 16px (`--space-4`) |
| Tablet/Desktop (≥48em) | `columns_desktop` (2–4) | 24px (`--space-6`) |

Typography scales across breakpoints:
- Banner title: `2xl` → `3xl` → `4xl`
- Section heading: Responsive based on `heading_size` setting

## Empty State

If the collection has no products or no collection is selected, displays a placeholder message using the `placeholder` snippet with icon and translated message.

## Accessibility

- Banner overlay text uses high-contrast white text on semi-transparent black background
- All images include proper `alt` attributes (collection title for banner)
- Focus states applied to all interactive elements
- Grid uses CSS Grid for automatic reflow on smaller viewports
- `prefers-reduced-motion` respected for transitions

## Performance Considerations

- CSS is scoped to `@layer sections` for cascade control
- Inline styles use CSS custom properties for dynamic values
- First 4 products load eagerly for LCP optimization
- Banner image optimized with modern `srcset` and `sizes`
- No external JavaScript dependencies
