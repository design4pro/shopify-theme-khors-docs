## Location

`snippets/price.liquid`

## LiquidDoc

```liquid
{% doc %}
  Renders product or variant pricing with support for sale prices, compare-at prices, unit pricing, and selling plans.
  Handles all Shopify pricing scenarios including price ranges, discounts, and subscriptions.

  @param {product} [product] - Product object for automatic price detection
  @param {variant} [variant] - Variant object for specific variant pricing (overrides product)
  @param {number} [price] - Manual price in cents (e.g., 1999 for $19.99)
  @param {number} [compare_at_price] - Manual compare-at price in cents
  @param {boolean} [show_unit_price] - Display unit pricing if available (default: false)
  @param {boolean} [show_badge] - Show "Sale" badge when on sale (default: false)
  @param {string} [size] - Size modifier: 'price--small', 'price--medium', 'price--large' (default: 'price--medium')
  @param {string} [class] - Additional CSS classes to append
  @param {string} [aria_label] - Custom ARIA label for accessibility

  @example Product with automatic price detection
    {% render 'price', product: product %}

  @example Variant with unit price
    {% render 'price', variant: variant, show_unit_price: true %}

  @example Manual pricing
    {% render 'price', price: 1999, compare_at_price: 2999 %}

  @example Large size with sale badge
    {% render 'price', product: product, size: 'price--large', show_badge: true %}

  @example Small size for product cards
    {% render 'price', variant: current_variant, size: 'price--small' %}
{% enddoc %}
```

## Usage

```liquid
{% comment %} Product with automatic price detection {% endcomment %}

{% comment %} Variant with unit price {% endcomment %}

{% comment %} Large size with sale badge {% endcomment %}

  product: product,
  size: 'price--large',
  show_badge: true

```

## Accessibility

- `role="region"` with `aria-label` on the price container for screen reader context
- `.visually-hidden` labels announce the price type (regular, sale, compare) to screen readers
- Sale badge uses `aria-hidden="true"` since price type is conveyed via visually-hidden text
- Price type (regular, sale, compare) is announced distinctly for assistive technology
