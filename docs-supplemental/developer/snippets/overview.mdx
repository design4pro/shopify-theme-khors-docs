## What are Snippets?

Snippets are reusable code fragments that can be included in sections, blocks, or other snippets using the `{% render %}` tag.

## LiquidDoc Requirement

All snippets must include a `{% doc %}` header documenting their purpose and parameters:

```liquid
{% doc %}
  Description of what this snippet does.

  @param {type} param - Description of parameter
  @param {type} [optional] - Optional parameter

  @example
  {% render 'snippet-name', param: value %}
{% enddoc %}
```

## Rendering Snippets

```liquid
{% render 'image',
  image: product.featured_image,
  width: 400,
  lazy: true
%}
```
