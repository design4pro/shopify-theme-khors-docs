## Location

`snippets/pagination.liquid`

## LiquidDoc

```liquid
{% doc %}
  Renders a pagination navigation component with numbered pages, previous/next arrows, and smart truncation.
  Uses Shopify's native paginate object to display page links with accessibility support.

  @param {object} paginate - Shopify paginate object (required)
  @param {string} [class] - Additional CSS classes to append
  @param {boolean} [show_page_numbers] - Display numbered page links (default: true)
  @param {boolean} [show_arrows] - Display previous/next arrow buttons (default: true)

  @example Basic pagination
    {% paginate collection.products by 12 %}
      {% render 'pagination', paginate: paginate %}
    {% endpaginate %}

  @example Pagination without page numbers
    {% paginate collection.products by 12 %}
      {% render 'pagination', paginate: paginate, show_page_numbers: false %}
    {% endpaginate %}

  @example Pagination with custom class
    {% paginate collection.products by 12 %}
      {% render 'pagination', paginate: paginate, class: 'pagination--compact' %}
    {% endpaginate %}
{% enddoc %}
```

## Usage

```liquid
{% comment %} Basic pagination for collection products {% endcomment %}
{% paginate collection.products by 12 %}
  {% render 'pagination', paginate: paginate %}
{% endpaginate %}

{% comment %} Arrows only (no page numbers) {% endcomment %}
{% paginate collection.products by 12 %}
  {% render 'pagination',
    paginate: paginate,
    show_page_numbers: false
  %}
{% endpaginate %}

{% comment %} With custom class for styling overrides {% endcomment %}
{% paginate collection.products by 12 %}
  {% render 'pagination',
    paginate: paginate,
    class: 'pagination--compact'
  %}
{% endpaginate %}
```

## CSS Classes

| Class | Description |
|---|---|
| `.pagination` | Main nav container |
| `.pagination__list` | UL list |
| `.pagination__item` | List item |
| `.pagination__item--arrow` | Arrow item |
| `.pagination__item--ellipsis` | Ellipsis item |
| `.pagination__link` | Link/button element |
| `.pagination__link--current` | Active page |
| `.pagination__link--disabled` | Disabled state |
| `.pagination__link--previous` | Previous arrow |
| `.pagination__link--next` | Next arrow |
| `.pagination__ellipsis` | Truncation dots |

## Accessibility

- `<nav>` with `aria-label` identifies the navigation landmark for screen readers
- `aria-current="page"` on the current page link communicates active state
- `aria-label` on all page links and arrows provides descriptive text (e.g., "Page 2", "Previous page")
- `aria-disabled="true"` on disabled arrows prevents interaction and announces disabled state
- `aria-hidden="true"` on ellipsis dots hides decorative truncation indicators from screen readers
- Smart truncation pattern (1 ... prev current next ... last) activates for 5+ pages to keep navigation manageable
- Focus-visible outlines on all interactive elements for keyboard navigation
