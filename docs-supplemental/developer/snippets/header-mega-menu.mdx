## LiquidDoc

```liquid
{% doc %}
  Renders a full-width mega menu panel for desktop navigation.
  Replaces the standard narrow dropdown for Level 1 items with children.
  Displays Level 2 items as column headings with Level 3 children beneath,
  plus an optional featured image slot from the link's associated object.

  Desktop only (hidden below 750px). Mobile navigation handled by header-drawer.

  @param {object} link - Level 1 Shopify link object with nested links
{% enddoc %}
```

## Usage

```liquid
{% comment %} Called internally by header-nav.liquid {% endcomment %}
{% render 'header-mega-menu', link: link %}
```

## JavaScript

The `MegaMenuNavigation` class (inline via `{% javascript %}`) provides keyboard navigation within the mega menu:

- Arrow Up/Down to move between menu items
- Arrow Up from first item returns focus to L1 button
- Home/End to jump to first/last item

The open/close behavior is handled by the parent `HeaderNavigation` class in `header-nav.liquid`.

## Accessibility

- `role="menu"` on container, `role="menuitem"` on all links
- `aria-current="page"` on current page links
- Active state highlighting
- Featured image link is `aria-hidden="true"` with `tabindex="-1"` (decorative)
- `focus-visible` outlines on all interactive elements
- Hidden on mobile (≤749px)
- `prefers-reduced-motion` disables fade transition
