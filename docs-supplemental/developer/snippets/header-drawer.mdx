  Renders a mobile navigation drawer with 3-level accordion menu, overlay backdrop,

## JavaScript

The `HeaderDrawer` class (inline via `{% javascript %}`) provides:

- **Open/close**: Uses native `dialog.showModal()` / `dialog.close()` for built-in focus trap
- **Trigger**: Listens to `[data-header-drawer-open]` button in header
- **Close methods**: Close button, overlay click, Escape key (via `cancel` event)
- **Swipe to close**: Touch events detect left-swipe > 80px on panel
- **Resize handler**: Auto-closes drawer if viewport expands past 750px
- **Focus restoration**: Returns focus to trigger button on close
- **Private fields**: `#dialog`, `#panel`, `#overlay`, `#closeBtn`, `#triggerBtn`

## Accessibility

- Native `<dialog>` provides built-in focus trap and `Escape` handling
- `aria-label` on dialog element with translation key
- Close button has `aria-label` for screen readers
- `aria-current="page"` on current page links
- Active page highlighting via Liquid link states
- All touch targets meet 44Ă—44px minimum
- `prefers-reduced-motion` disables all slide/fade transitions
- Hidden on desktop (â‰Ą750px) via `display: none`
