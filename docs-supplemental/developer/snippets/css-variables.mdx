## Location

`snippets/css-variables.liquid`

## LiquidDoc

```liquid
{% doc %}
  Outputs CSS custom properties from theme settings.
  Include in theme.liquid inside <style> tag.

  @example
  <style>
    :root {
      {% render 'css-variables' %}
    }
  </style>
{% enddoc %}
```

## Generated Variables

```css
:root {
  /* Colors */
  --color-primary: #1A1A1A;
  --color-secondary: #F5A623;
  --color-background: #FFFFFF;
  --color-text: #333333;

  /* Typography */
  --font-heading: "Playfair Display", serif;
  --font-body: "Open Sans", sans-serif;
  --font-size-base: 16px;

  /* Spacing */
  --spacing-section: 60px;
  --spacing-grid: 20px;

  /* Layout */
  --page-width: 1200px;
  --border-radius: 4px;
}
```

## Usage in CSS

```css
.button {
  background-color: var(--color-primary);
  font-family: var(--font-body);
  border-radius: var(--border-radius);
}
```
