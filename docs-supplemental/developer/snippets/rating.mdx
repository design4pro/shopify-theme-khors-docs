## Location

`snippets/rating.liquid`

## LiquidDoc

```liquid
{% doc %}
  Renders a star rating component with support for partial stars, review count, and multiple sizes.
  Uses inline SVG stars with fill for color control (not the icon system which uses stroke).

  @param {number} rating - Rating value from 0 to 5, supports decimals (default: 0)
  @param {number} [max_rating] - Maximum rating value (default: 5)
  @param {number} [review_count] - Number of reviews; hidden if empty
  @param {boolean} [show_value] - Show numeric rating value beside stars (default: false)
  @param {string} [size] - Size modifier: 'rating--small', 'rating--medium', 'rating--large' (default: 'rating--medium')
  @param {string} [star_color] - Custom star color as CSS value; defaults to gold (#D4A017)
  @param {string} [class] - Additional CSS classes

  @example Basic rating
    {% render 'rating', rating: 4.5 %}

  @example Rating with review count
    {% render 'rating', rating: 4.5, review_count: 128 %}

  @example Small rating with numeric value
    {% render 'rating', rating: 3.0, show_value: true, size: 'rating--small' %}

  @example Custom star color
    {% render 'rating', rating: 5, star_color: '#ff6600' %}
{% enddoc %}
```

## CSS Classes

| Class | Description |
|-------|-------------|
| `.rating` | Base component (inline-flex, default gold star color) |
| `.rating--small` | Small size (14px stars, xs font) |
| `.rating--medium` | Medium size (18px stars, sm font, default) |
| `.rating--large` | Large size (24px stars, base font) |
| `.rating__stars` | Stars container (flex row with 1px gap) |
| `.rating__star` | Individual star wrapper |
| `.rating__star--full` | Filled star (uses `--rating-star-color`) |
| `.rating__star--half` | Half-filled star (relative positioned container) |
| `.rating__star--empty` | Empty star (uses `--rating-star-empty` border color) |
| `.rating__star-bg` | Half star background layer (empty color) |
| `.rating__star-fill` | Half star fill layer (50% width, overflow hidden) |
| `.rating__value` | Numeric rating display (semibold) |
| `.rating__count` | Review count display (muted color) |

## Accessibility

- `role="img"` on the container element for semantic meaning
- `aria-label` with full rating text (e.g., "4.5 out of 5 stars" or "4.5 out of 5 stars, 128 reviews")
- Stars container marked `aria-hidden="true"` to prevent individual star announcement
- Partial stars rendered with 50% clip technique (overflow hidden on `.rating__star-fill`)
- Rating value rounded to nearest 0.5 for consistent star display
