<Info>
  Access: **Shopify Admin > Online Store > Themes > Customize > Theme Settings > Layout**
</Info>

## Layout Settings

### Page Width
Set the maximum width of your store content.

### Section Spacing
Control vertical spacing between sections.

### Container Padding
Adjust horizontal padding on mobile and desktop.

## Grid Settings

Configure product grid columns and gaps for collection pages.
