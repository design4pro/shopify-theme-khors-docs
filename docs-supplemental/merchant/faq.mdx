  <Accordion title="How do I install Khors Theme?">
    Follow our [installation guide](/merchant/installation) to add Khors to your store.
  </Accordion>

  <Accordion title="Can I switch between presets?">
    Yes, you can switch presets at any time from Theme Settings. Note that switching presets will reset some customizations.
  </Accordion>

    Yes, Khors is fully responsive and optimized for all screen sizes.
  </Accordion>

  <Accordion title="Does Khors support multiple languages?">
    Yes, Khors supports Shopify's translation system for multi-language stores.
  </Accordion>

    Visit our [support page](/merchant/support) for contact information.
