## Overview

The collection section displays products in a grid with filtering and sorting options.

## Settings

### Products Per Row
Set the number of products per row (2-5).

### Products Per Page
Configure pagination settings.

### Filters
Enable sidebar or horizontal filters.

### Sorting
Allow customers to sort by price, date, popularity.
