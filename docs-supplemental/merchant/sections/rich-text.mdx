## Overview

The Rich Text section is your go-to for adding formatted content to any page. Use it for about stories, policies, product care instructions, or any text that needs headings, bold/italic formatting, lists, and links.

## Settings

### Content

| Setting | Description |
|---------|-------------|
| **Content** | Rich text editor with formatting toolbar. Supports headings, bold, italic, links, and lists. |

### Layout

| Setting | Description |
|---------|-------------|
| **Width** | Narrow (best for reading), Medium (balanced), or Full (spans page width) |
| **Text alignment** | Left, Center, or Right |

### Style

| Setting | Description |
|---------|-------------|
| **Color scheme** | Light, Dark, or Accent background color |

### Spacing

| Setting | Description |
|---------|-------------|
| **Padding top/bottom** | Adjust vertical spacing from 0 to 100px |

## Using the Rich Text Editor

The content editor includes these formatting options:

| Button | Function |
|--------|----------|
| **H2–H6** | Add headings (H2 largest, H6 smallest) |
| **B** | Bold text |
| **I** | Italic text |
| **U** | Underline text |
| **Link icon** | Add hyperlinks to pages or external sites |
| **Bullet list** | Create unordered list |
| **Number list** | Create numbered list |

<Tip>
Use headings (H2, H3, H4) to structure your content. This improves readability and helps with SEO.
</Tip>

## Width Guidelines

<Tabs>
  <Tab title="Narrow (800px)">
    **Best for:**
    - Long-form reading (about stories, blog posts)
    - Legal text (shipping policy, privacy policy)
    - Maximum readability for paragraph-heavy content

    **Text alignment:** Left recommended
  </Tab>
  <Tab title="Medium (1000px)">
    **Best for:**
    - About pages with moderate content
    - Product care instructions
    - FAQ sections
    - General informational content

    **Text alignment:** Left or Center
  </Tab>
  <Tab title="Full Width">
    **Best for:**
    - Promotional messaging
    - Short, centered statements
    - Landing page headlines
    - Multi-column content (when paired with other sections)

    **Text alignment:** Center recommended
  </Tab>
</Tabs>

## Common Use Cases

<AccordionGroup>
  <Accordion title="About Page Story">
    **Settings:**
    - Width: Medium
    - Alignment: Left
    - Color scheme: Light

    **Content tips:**
    - Start with H2 heading ("Our Story")
    - Use 2-4 paragraphs of 3-5 sentences each
    - Add H3 subheadings for brand values
    - Include links to product collections or team page
  </Accordion>

  <Accordion title="Shipping Policy">
    **Settings:**
    - Width: Narrow
    - Alignment: Left
    - Color scheme: Light

    **Content tips:**
    - Use H2 for main title, H3 for sections
    - Create numbered lists for shipping options
    - Bold important terms (delivery times, costs)
    - Link to FAQ or contact page for questions
  </Accordion>

  <Accordion title="Landing Page Hero Text">
    **Settings:**
    - Width: Full
    - Alignment: Center
    - Color scheme: Accent
    - Padding: 80px top and bottom

    **Content tips:**
    - Keep it short (1-2 paragraphs max)
    - Use H2 for main headline
    - Bold key benefits or offers
    - Include call-to-action link
  </Accordion>

  <Accordion title="Product Care Guide">
    **Settings:**
    - Width: Medium
    - Alignment: Left
    - Color scheme: Light

    **Content tips:**
    - H2 title ("Care Instructions")
    - Use bulleted lists for washing/care steps
    - Bold warning text ("Do not bleach")
    - Link to related products or warranty info
  </Accordion>
</AccordionGroup>

## Best Practices

<Check>Break up long text with headings every 2-3 paragraphs for scanability</Check>

<Check>Use Narrow width for text-heavy content to prevent eye strain from long line lengths</Check>

<Check>Keep paragraphs to 3-5 sentences for easy mobile reading</Check>

<Check>Add links to related pages or products to keep visitors engaged</Check>

<Warning>
Avoid pasting content from Word or Google Docs. This can bring hidden formatting. Instead, paste as plain text and re-apply formatting in Shopify.
</Warning>

<Tip>
For content with images, use the "Image with Text" section instead. Rich Text is text-only.
</Tip>

## Combining with Other Sections

The Rich Text section works great alongside:

- **Image with Text** — Alternate text blocks with visuals
- **Featured Collection** — Add context above product grids
- **Newsletter** — Include informational text before signup form
- **Slideshow** — Follow hero images with detailed text

## SEO Tips

- Use **one H2** per section as the main topic
- Include **keywords naturally** in headings and first paragraph
- Add **internal links** to related products or collections
- Keep **total word count** between 150-500 words for featured sections

## Accessibility

The section automatically handles:
- Proper heading hierarchy (H2 → H3 → H4)
- Sufficient color contrast for text
- Adequate line spacing for readability
- Keyboard-accessible links
