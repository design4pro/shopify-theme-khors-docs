<Info>
  Access: **Shopify Admin > Online Store > Themes > Customize > Add section**
</Info>

## Overview

Custom sections allow you to add flexible content blocks to any page.

## Available Section Types

- **Rich Text** - Text with formatting options
- **Image with Text** - Side-by-side layout
- **Featured Collection** - Showcase products
- **Video** - Embed video content
- **Slideshow** - Image carousel
- **Multicolumn** - Grid of content blocks
