## Overview

The footer appears at the bottom of every page. It supports multiple content columns (menus and text blocks), an email newsletter signup, social media links, payment method icons, and a copyright notice.

## Content Blocks

Add blocks to create footer columns:

### Menu Block

Add navigation menus as columns. Each block includes:
- **Heading** — Column title (e.g., "Quick Links", "Customer Service")
- **Menu** — Select a navigation menu to display

### Text Block

Add custom text content as columns. Each block includes:
- **Heading** — Column title (e.g., "About Us")
- **Content** — Rich text editor for custom content

## Settings

| Setting | Description |
|---------|-------------|
| **Show newsletter** | Display email signup form (default: on) |
| **Newsletter heading** | Text above the signup form (default: "Stay in touch") |
| **Show social links** | Display social media icons (default: on) |
| **Show payment icons** | Display accepted payment method badges (default: on) |
| **Show copyright** | Display copyright notice with your store name (default: on) |
| **Show border** | Top border line (default: on) |
| **Color scheme** | Light, dark, or accent color scheme |
| **Padding top/bottom** | Adjust vertical spacing from 0 to 80px |

<Note>
Social media links are configured in **Theme Settings > Social media**, not in the footer section itself. The footer reads these global settings automatically.
</Note>

## Industry Tips

<Tabs>
  <Tab title="Fashion">
    Use 3–4 menu columns (Collections, About, Help, Legal) with a text block for your brand story. Enable newsletter and social links to build community.
  </Tab>
  <Tab title="Beauty">
    Include a "Skin Type Guide" or "Ingredient Glossary" menu alongside standard navigation. The newsletter signup helps capture leads for product launches.
  </Tab>
  <Tab title="Electro">
    Add "Technical Support" and "Warranty Information" menu columns. Payment icons build trust for high-value purchases. Consider hiding the newsletter if you use a dedicated popup.
  </Tab>
</Tabs>
