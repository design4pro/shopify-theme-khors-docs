## Overview

The Featured Collection section displays a selection of products from any collection in your store. Perfect for highlighting bestsellers, new arrivals, or seasonal products on your homepage or landing pages.

## Settings

### Collection Selection

| Setting | Description |
|---------|-------------|
| **Collection** | Choose which collection to feature from your store |

<Tip>
Create collections in **Products > Collections** to organize products by category, season, or promotion. The featured collection will automatically update when you add or remove products from the selected collection.
</Tip>

### Banner

| Setting | Description |
|---------|-------------|
| **Show banner** | Display the collection's featured image as a banner above products |
| **Banner aspect ratio** | Choose from Square (1:1), Portrait (3:4), Landscape (4:3), or Wide (16:9) |
| **Show banner overlay** | Add a semi-transparent overlay with collection title and description |
| **Overlay opacity** | Light, Medium, or Dark overlay darkness |

<Note>
The banner uses the collection's featured image. Set this in **Products > Collections > [Your Collection] > Collection image**.
</Note>

### Content

| Setting | Description |
|---------|-------------|
| **Heading** | Custom section title (defaults to "Featured collection") |
| **Heading size** | Choose from H2 (largest), H3, or H4 (smallest) |
| **Description** | Optional text below the heading (limited to 200 characters) |
| **Text alignment** | Left, Center, or Right alignment for heading and description |

### Product Display

| Setting | Description |
|---------|-------------|
| **Products to show** | Number of products to display, from 4 to 12 |
| **Desktop columns** | Grid layout on desktop: 2, 3, or 4 columns |
| **Mobile columns** | Grid layout on mobile: 1 or 2 columns |
| **Product image ratio** | Square, Portrait, or Landscape product images |
| **Show secondary image** | Display alternate image on hover (uses second product image) |
| **Show badges** | Display "Sale", "New", or custom badges on products |
| **Show rating** | Display product star ratings (requires review app) |
| **Show swatches** | Display color/variant swatches below product |
| **Enable quick add** | Show "Add to Cart" button directly on product cards |
| **Card style** | Default (with border) or Minimal (borderless) |

### View All Button

| Setting | Description |
|---------|-------------|
| **Show view all** | Display button linking to full collection page |
| **Button text** | Customize button text (default: "View all") |

### Spacing

| Setting | Description |
|---------|-------------|
| **Padding top/bottom** | Adjust vertical spacing from 0 to 100px |

## Industry Best Practices

<Tabs>
  <Tab title="Fashion">
    - Use **4 columns desktop** with **Portrait image ratio** for editorial look
    - Enable **Show secondary image** for outfit inspiration on hover
    - Enable **Show swatches** to display available colors
    - Use **Landscape banner** with overlay for seasonal campaigns
  </Tab>
  <Tab title="Beauty">
    - Use **3 columns desktop** with **Square image ratio** for clean product focus
    - Enable **Show badges** to highlight "New" or "Bestseller" products
    - Enable **Show rating** to build trust with customer reviews
    - Use **Wide banner (16:9)** for lifestyle imagery
  </Tab>
  <Tab title="Electro">
    - Use **4 columns desktop** with **Landscape image ratio** for product details
    - Enable **Quick add** for fast purchasing
    - Enable **Show rating** for product comparisons
    - Use **Square banner** or disable banner for spec-focused display
  </Tab>
</Tabs>

## Common Use Cases

<AccordionGroup>
  <Accordion title="Homepage New Arrivals">
    Create a "New Arrivals" collection (automated by publish date), feature it with banner enabled and "View all" button. Update heading to "Just In" or "New This Week".
  </Accordion>

  <Accordion title="Bestsellers Section">
    Create a manual collection of top sellers, disable banner for clean focus, use 4 columns desktop with product ratings enabled.
  </Accordion>

  <Accordion title="Seasonal Promotion">
    Feature seasonal collection with wide banner (16:9), overlay enabled with promotional description, and custom "Shop [Season]" button text.
  </Accordion>

  <Accordion title="Category Preview">
    Add multiple featured collection sections to homepage, each showing 4-8 products from different categories (e.g., "Dresses", "Accessories", "Shoes"). Use centered text alignment and view all buttons.
  </Accordion>
</AccordionGroup>

## Tips

<Check>Enable secondary images by uploading multiple product photos — the second image shows on hover for added visual interest</Check>

<Check>Set products count to match your column count (e.g., 4 or 8 products for 4-column layout) for balanced grid appearance</Check>

<Warning>
If your collection has fewer products than the "Products to show" setting, the section will only display available products without empty spaces.
</Warning>

<Tip>
For best performance, optimize collection featured images to at least 2400px wide for sharp display on large screens.
</Tip>
