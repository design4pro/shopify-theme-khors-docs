## Overview

The header appears at the top of every page and includes your logo, main navigation menu, search, account link, and cart icon. It supports three layout styles and can stick to the top of the page as visitors scroll.

## Settings

### Logo

| Setting | Description |
|---------|-------------|
| **Desktop logo** | Upload your main logo image (recommended: SVG or PNG with transparent background) |
| **Mobile logo** | Optional separate logo for mobile devices (falls back to desktop logo) |
| **Desktop logo width** | Logo width on desktop, from 80px to 300px (default: 150px) |
| **Mobile logo width** | Logo width on mobile, from 60px to 200px (default: 100px) |
| **Logo alignment** | Left, center, or right alignment |
| **Logo alt text** | Custom alt text for accessibility (defaults to your store name) |

### Navigation

| Setting | Description |
|---------|-------------|
| **Menu** | Select your main navigation menu (supports up to 3 levels of nesting) |

### Layout

| Setting | Description |
|---------|-------------|
| **Layout** | Choose from 3 styles: Logo left / Nav center (default), Logo center / Nav below, or Logo left / Nav left |
| **Sticky header** | Header stays visible at the top when scrolling (default: on) |
| **Transparent header** | Header background becomes transparent (for hero sections) |
| **Show search** | Display search icon in the header (default: on) |
| **Show account** | Display account icon (only visible when customer accounts are enabled) |

### Style

| Setting | Description |
|---------|-------------|
| **Color scheme** | Light, dark, or accent color scheme |
| **Show border** | Bottom border line (default: on) |
| **Padding top/bottom** | Adjust vertical spacing from 0 to 40px |

## Industry Tips

<Tabs>
  <Tab title="Fashion">
    Use the **Logo center / Nav below** layout with a generous logo width (200–250px) for a clean, editorial look. Enable the transparent header for hero banners on the homepage.
  </Tab>
  <Tab title="Beauty">
    Choose **Logo left / Nav center** layout with mega menus to showcase product categories with featured collection images. Keep the logo width moderate (120–150px).
  </Tab>
  <Tab title="Electro">
    Use **Logo left / Nav left** layout to maximize space for search. The predictive search feature helps customers find products quickly by name or category.
  </Tab>
</Tabs>
